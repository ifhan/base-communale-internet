Comment lire et télécharger les vidéos ?

Sélectionner un élément dans la liste ci-dessous. Deux possibilités sont offertes :

  • Lire les vidéos en streaming : les vidéos ne sont accessibles qu'avec des versions récentes des navigateurs web Mozilla Firefox, Google Chrome, Opera.
  • Télécharger les vidéos : une fois téléchargées, les vidéos sont lisibles à l'aide du logiciel libre VLC.

Pour télécharger les vidéos, cliquer sur le lien Télécharger avec le bouton droit de la souris puis sélectionner dans le menu contextuel "Enregistrer sous" ou "Enregistrer la cible du lien sous".
